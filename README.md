# Assignment 01 – Web Development (Figma to HTML/CSS)

This was my first assignment in the web development course, where I converted a given Figma design into a static webpage using only **HTML** and **CSS**.

## Features

* Hero/banner section
* Multiple content sections
* Motivation and inspiration sections
* Embedded YouTube video
* Learning roadmap (Current & Next Targets)
* Simple contact/support form
* Footer with social links

## Technologies Used

* HTML5
* CSS3
* Google Fonts
